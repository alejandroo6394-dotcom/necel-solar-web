# Logotipos de fabricantes

Logotipos de las marcas con las que trabaja Necel Solar, recortados de la
imagen aportada por el cliente (fondo transparente):

- `fischer.png`
- `longi.png` (LONGi Solar)
- `bosch.png`
- `necel.png` (Necel Solar)
- `growatt.png`

Todos comparten la misma altura de lienzo para conservar su tamaño relativo
en la franja de marcas. También se incluye `marcas-oficiales.png` con la
franja completa original por si se quiere usar como una sola imagen.

Para actualizar un logotipo, sustituye el archivo manteniendo el mismo nombre
o cambia la ruta en `index.html` dentro de la sección `#marcas`.
