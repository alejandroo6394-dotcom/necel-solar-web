# Logotipos de fabricantes

Estos archivos SVG son versiones monocromas provisionales para la franja
"Marcas colaboradoras". Están construidos con `currentColor`, por lo que la
hoja de estilos controla su color (gris en reposo, color de marca al pasar
el ratón).

Para usar los logotipos oficiales, sustituye cada archivo manteniendo el
mismo nombre (longi.svg, bosch.svg, growatt.svg, fischer.svg) o actualiza
las rutas en `index.html` dentro de la sección `#marcas`.
